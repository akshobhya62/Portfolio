Experience

Work Experience

AI Generated Jitter Buffer Delay Prediction Model and Optimization - Software Engineer 2022-2023

I spearheaded the development of an innovative Jitter Buffer Delay Prediction Model and its subsequent optimization. The project aimed to enhance the call quality experience by predicting and mitigating delays introduced by jitter in network-transmitted call packets. I leveraged deep learning models, including Convolutional Neural Networks (CNNs) and Transformers, experimenting to identify the most effective architecture. Notably, I implemented the model from scratch in C++, ensuring efficient execution on Samsung's upcoming flagship S24 smartphone. The optimization phase involved utilizing OpenCL and SIMD/Arm-Neon to significantly reduce the inference time, achieving a remarkable 22x improvement from the initial 4.3ms to a highly efficient 0.2ms. This work represents a substantial contribution to Samsung's telecommunications technology, promising a seamless and enhanced user experience in future devices.

Mission Critical Services(MCPTT/MCVideo/MCData) - Software Engineer 2022-2023

I played a key part in the groundbreaking development of the world's first commercially available Mission-Critical Push-to-Talk (MCPTT) solution on Android. This innovative solution, aligned with the 3GPP MCX communication standard, was tailored for first responders, introducing a new era in mission-critical communication technology. As a project lead, I led the conception and implementation of essential features such as Call Forwarding and Remote Monitoring, ensuring strict adherence to 3GPP specifications and customized requirements. The design and development process included the integration of adhoc file distribution (FD) functionality into Samsung's mission-critical Push-to-Talk (PTT) app middleware, accomplished through proficient use of Java and C++. Rigorous testing and validation, facilitated by comprehensive JUnit Test Cases, validated the robustness and reliability of these critical features, contributing to the overall success of the mission-critical communication solution.

Deep Learning Model Compression using Knowledge distillation - Software Engineer Intern 2022-2023

In my role as a Software Engineer Intern at Samsung Research, I undertook a significant project focused on Deep Learning Model Compression using Knowledge Distillation. The project involved compressing the Retinaface model, a state-of-the-art facial detection model, by employing knowledge distillation techniques. I meticulously implemented both the teacher model and the student model, reducing the student model's parameters significantly while maintaining its performance. Leveraging the WIDERFACE dataset for training, the compressed model achieved impressive results, boasting test accuracies of 90.3%, 88.7%, and 83.9% for easy, medium, and hard test sets, respectively. This project not only demonstrated my proficiency in deep learning frameworks such as PyTorch and TensorFlow but also showcased my ability to optimize model architectures, a crucial skill in deploying efficient and resource-friendly solutions in real-world applications.
